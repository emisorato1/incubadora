Proceso de incubacion huevos los cuales se dividen en tres tipos de huevos, 
pasan por dos procesos el primero de incubación y el segundo de nacedoras. 

En el primer proceso de incubación el usuario coloca cierta cantidad de huevos 
de los tres tipos de gallinas, además de la fecha de inicio de la incubación, y la 
icubadora utilizada, se coloca en alguna de las 6 incubadoras, donde hay 3 modernas y 
3 antiguas. Mientras sucede el proceso de incubación de las incubadoras se registra la 
información de un sensor de motor de volteado, tres sensores de ventilación, un sensor 
de temperatura, un sensor de humedad y un sensor de la resistencia de evaporación. 
Al final del proceso el usuario coloca la fecha final, la cantidad de huevos fértiles, y
a la nacedora a la que se dirige.

En el segundo proceso de nacedoras el usuario coloca cierta cantidad de huevos 
fértiles de cada una, además de la fecha de inicio de la nacedora y de que incubadora viene, 
se coloca en alguna de las 4 nacedoras, donde hay 2 modernas y 2 antiguas. Mientras 
sucede el proceso de incubación de las incubadoras se registra la información de tres 
sensores de ventilación, un sensor de temperatura, un sensor de humedad y un sensor 
de la resistencia de evaporación. Al final del proceso el usuario coloca la fecha final 
del proceso, la cantidad de nacimientos y descate de belichas machos.

Entidades principales:
Usuario: Persona que realiza el proceso de incubación y nacimiento.
Incubadora: Dispositivo donde se lleva a cabo el proceso de incubación. Se distinguen entre incubadoras modernas y antiguas.
Nacedora: Dispositivo donde se lleva a cabo el proceso de nacimiento. También se distinguen entre modernas y antiguas.
Sensor: Dispositivo que registra información durante el proceso de incubación y nacimiento.

Relaciones:
Un Usuario puede realizar muchos procesos de incubación y nacimiento.
Un Incubadora puede ser utilizada en muchos procesos de incubación y nacimiento.
Un Nacedora puede ser utilizada en muchos procesos de nacimiento.
Muchos Sensores están asociados con cada proceso de incubación y nacimiento.

Instalacion de postgres en docker por consola

1- Este comando crea la base de datos con los datos que le demos
 docker run --name my_postgres -e POSTGRES_USER=cardozo -e POSTGRES_PASSWORD=12345 -e POSTGRES_DB=db_inta -p 5432:5432 -d postgres:15.6
 -d: sirve para que podamos cerrar la consola (ejecutar en segundo plano)
 -p 5432:5432: Mapea el puerto 5432 del contenedor al puerto 5432 del host (localhost). Esto permite que puedas 
                conectarte a PostgreSQL desde tu máquina local
 Comando para ver si esta en proceso la base de datos: docker ps -a
 Comando para parar la db: docker stop name(nombre que le asignamos como proceso "my_postgres")
 Comando para poder eliminar la db: docker rm name(nombre que le asignamos como proceso "my_postgres") 
 Comando para verifica los logs del contenedor: docker logs my_postgres
 Comando para habilitar la apertura de la base de datos en la consola: docker exec -it my_postgres bash
 Comando para abrir la base de datos en la consola: psql -U cardozo --db db_inta --password
 Comando para ver las tablas y relaciones que hay: \d
